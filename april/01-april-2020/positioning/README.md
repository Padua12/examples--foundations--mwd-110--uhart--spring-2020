# Positioning

## Offset Properties

`top`, `right`, `bottom`, and `left`.

Example

```css
h1 {
   top: 10px;
   left: 30px;
}

header {
   bottom: -30px;
   right: -400px;
}

```

---

## Static
* Default/initial value to positioned items
* Doesn’t break flow
* Doesn’t observe offset properties
* `z-index` (stacking context) isn’t observed
